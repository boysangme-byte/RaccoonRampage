# 🦝 Raccoon Rampage

3D FPS 게임 - HTML 파일 하나로 실행 가능!

## 🎮 게임 실행 방법

1. `game.html` 파일을 더블클릭하거나
2. 웹 브라우저로 `game.html` 파일을 열기

**주의**: 사운드 파일(.mp3)은 포함되어 있지 않습니다. 사운드 없이도 게임은 정상 작동합니다.

## 🎯 게임 조작법

- **WASD**: 이동
- **마우스**: 시점 조작
- **Space**: 점프
- **Shift**: 달리기
- **마우스 왼쪽 클릭**: 발사
- **R**: 재장전

## 🛠️ 기술 스택

- Three.js (3D 그래픽)
- HTML5 Canvas
- JavaScript (ES6+)

## 📝 파일 구조

- `game.html`: 모든 코드가 번들링된 단일 실행 파일
- `index.html`: 원본 모듈 버전 (개발용)
- `js/`: 원본 JavaScript 모듈 파일들
- `css/`: 원본 CSS 파일

## 🚀 GitHub Pages로 배포하기

1. GitHub 저장소 생성
2. `game.html` 파일 업로드
3. Settings > Pages에서 `game.html`을 index로 설정
4. 또는 `game.html`을 `index.html`로 이름 변경
